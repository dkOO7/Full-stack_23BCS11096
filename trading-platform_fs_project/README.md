# Trading Platform Project

This project is a full-stack trading platform application. The project is divided into several modules, each representing a stage in the development lifecycle.

## Directory Structure

- `working_project/`: This directory contains the original, untouched working project.
- `module1/`: This module contains the initial setup and core logic of the backend (Spring Boot).
- `module2/`: This module is planned for the implementation of authentication and security features for the backend.
- `module3/`: This module contains the initial setup and basic components of the frontend (React).
- `module4/`: This module is planned for the implementation of advanced UI features and pages for the frontend.
- `module5/`: This module is designated for deployment-related configurations and scripts.
  - `deployment/`: Contains deployment scripts and configurations.
